We are using py.test for Testdrivedevelopment. A simple beginner 
Tutorial you could find here: http://yourpart.eu/p/pycamp2013-pytest

to run a test you need to have py.test installed 
( pip install pytest )
to call a test:
py.test-2.7 test_namespaces.py
(verbouse mode)
py.test-2.7 -v test_namespaces.py
